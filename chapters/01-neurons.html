⚡ What is a Neuron?

The neuron is the fundamental building block of every neural network. Understanding how a single neuron works unlocks everything else.

The Weighted Voting Machine

Imagine you're deciding whether to go to a party. You consider several factors: Is it raining? (x₁), Will my friends be there? (x₂). But these factors don't all matter equally — whether your friends will be there probably matters more than the rain. So you assign weights to each factor.

That's exactly what an artificial neuron does. It's a weighted voting machine: it takes multiple inputs, multiplies each by a weight (importance), sums everything up, adds a bias (a default inclination), and then applies an activation function to decide its output.

💡 Key Intuition:

A neuron doesn't just count votes — it weights them. High-weight inputs have more influence on the final decision than low-weight ones. The bias lets the neuron fire even when all inputs are zero.

The Math

For a neuron with two inputs x₁ and x₂:

z = w₁·x₁ + w₂·x₂ + b
output = activation(z)

Where:

  • x₁, x₂ — the input values (your data)
  • w₁, w₂ — the weights (learned during training)
  • b — the bias (a learnable offset)
  • z — the weighted sum (pre-activation)
  • activation(z) — a non-linear function applied to z

Activation Functions

Without an activation function, a neuron is just linear — no matter how many layers you stack, you'd only get a linear transformation. Activation functions introduce non-linearity, which is what gives neural networks their expressive power.

Sigmoid σ(z)

Outputs a value between 0 and 1. Great for probabilities. Formula: σ(z) = 1 / (1 + e^(-z))

ReLU (Rectified Linear Unit)

The most popular activation in modern networks. Simple: if z > 0, output z. Otherwise, output 0. Formula: ReLU(z) = max(0, z)

Tanh

Like sigmoid but centered at 0, outputting values between -1 and 1. Formula: tanh(z) = (e^z - e^(-z)) / (e^z + e^(-z))

⚠️ The Dying ReLU Problem:

If a neuron's weighted sum is always negative, ReLU outputs 0 forever — the neuron "dies" and stops learning. Variants like Leaky ReLU or GELU fix this.

Putting It Together

A single neuron can learn simple patterns — but it's limited. A neuron with sigmoid activation essentially learns a linear decision boundary. To solve complex problems, you need many neurons organized into layers — which is exactly what Chapter 2 covers.

✅ Key Takeaways:
  • A neuron computes a weighted sum of inputs plus a bias
  • Weights determine how much each input matters
  • Bias lets a neuron have a "default" lean
  • Activation functions add non-linearity — essential for learning complex patterns
  • ReLU is the most commonly used activation in modern networks
