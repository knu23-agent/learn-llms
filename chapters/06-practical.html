🔧 Practical Examples

Enough theory — let's write code. These examples use Hugging Face Transformers, the go-to library for working with LLMs in Python.

Setup: Install the required libraries first: 
pip install transformers torch

1. Text Generation with GPT-2

The pipeline API is the fastest way to use a model. This loads GPT-2 and generates text from a prompt:

from transformers import pipeline

# Download and load the model (first run downloads ~500MB)
generator = pipeline("text-generation", model="gpt2")

result = generator(
    "Once upon a time, a neural network",
    max_new_tokens=60,
    num_return_sequences=1,
    temperature=0.8,   # higher = more creative
    do_sample=True,
)

print(result[0]["generated_text"])

Key parameters:

  • max_new_tokens — how many tokens to generate
  • temperature — controls randomness. 0 = deterministic, 1 = standard, >1 = chaotic
  • do_sample=True — sample from the distribution instead of always picking the top token

2. Sentiment Analysis

Classification pipelines are even simpler. This uses a model fine-tuned on reviews:

from transformers import pipeline

classifier = pipeline("sentiment-analysis")

results = classifier([
    "I love learning about neural networks!",
    "This is incredibly confusing and frustrating.",
    "The model performs reasonably well overall.",
])

for text, result in zip(["..."]*3, results):
    print(f"{result['label']}: {result['score']:.2%}")

# Output:
# POSITIVE: 99.9%
# NEGATIVE: 99.7%
# POSITIVE: 97.3%

3. Word Embeddings with BERT

Embeddings are dense vector representations of text. BERT produces contextual embeddings — the same word gets a different vector depending on context:

from transformers import AutoTokenizer, AutoModel
import torch

tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")
model = AutoModel.from_pretrained("bert-base-uncased")

text = "Neural networks are fascinating."
inputs = tokenizer(text, return_tensors="pt")

with torch.no_grad():
    outputs = model(**inputs)

# last_hidden_state shape: [1, seq_len, 768]
# Each token → 768-dimensional vector
embeddings = outputs.last_hidden_state
print(embeddings.shape)

# Sentence embedding: mean-pool all token vectors
sentence_embedding = embeddings.mean(dim=1)  # [1, 768]
print(sentence_embedding.shape)

Embeddings are useful for:

  • Semantic search — find similar documents by comparing vectors
  • Clustering — group similar texts
  • Feature input — feed into downstream classifiers

4. Fine-Tuning a Classifier

Pre-trained models can be fine-tuned on your specific task with relatively little data. Here's a minimal fine-tuning loop:

from transformers import (
    AutoModelForSequenceClassification,
    AutoTokenizer,
    Trainer,
    TrainingArguments,
)
from datasets import load_dataset

# Load a pre-trained model with a classification head
model_name = "bert-base-uncased"
model = AutoModelForSequenceClassification.from_pretrained(
    model_name, num_labels=2  # positive / negative
)
tokenizer = AutoTokenizer.from_pretrained(model_name)

# Load and tokenize a dataset
dataset = load_dataset("imdb")
def tokenize(batch):
    return tokenizer(batch["text"], truncation=True, padding="max_length")

dataset = dataset.map(tokenize, batched=True)

# Training configuration
args = TrainingArguments(
    output_dir="./results",
    num_train_epochs=3,
    per_device_train_batch_size=16,
    evaluation_strategy="epoch",
    save_strategy="epoch",
    load_best_model_at_end=True,
)

trainer = Trainer(
    model=model,
    args=args,
    train_dataset=dataset["train"].shuffle(seed=42).select(range(2000)),
    eval_dataset=dataset["test"].select(range(500)),
)

trainer.train()
# Model is now fine-tuned on movie review sentiment
