🕸️ Neural Networks

A neural network is just many neurons organized into layers. Data flows forward through each layer — transformed at every step — until it reaches the output.

Layers: The Core Structure

Every neural network has at least three kinds of layers:

  • Input Layer — Raw data enters here. Each node is one feature (e.g., a pixel value).
  • Hidden Layer(s) — Where the magic happens. Neurons learn to detect patterns. Deep networks have many hidden layers — that's why they're called "deep" learning.
  • Output Layer — The prediction. For classification, each output neuron corresponds to a class. For regression, one neuron outputs a number.
💡 Why "Deep" Learning?

The depth refers to the number of layers. A shallow network might have 1-2 hidden layers. A deep network can have dozens or hundreds. GPT-4 has 96 layers!

The Forward Pass

The forward pass is how data moves through the network to produce a prediction. At each layer, every neuron:

  1. Receives outputs from all neurons in the previous layer
  2. Computes the weighted sum + bias: z = Wx + b
  3. Applies activation: a = activation(z)
  4. Passes its output a to all neurons in the next layer
Layer output: a⁽ˡ⁾ = activation(W⁽ˡ⁾ · a⁽ˡ⁻¹⁾ + b⁽ˡ⁾)

In matrix form, this is incredibly efficient — one matrix multiplication handles all the neurons in a layer simultaneously. This is why GPUs are essential: they're optimized for parallel matrix operations.

Fully Connected (Dense) Layers

In a fully connected layer, every neuron connects to every neuron in the next layer. This is the most basic type. Each connection has its own weight. A 3-neuron layer connecting to a 4-neuron layer has 3 × 4 = 12 weight parameters.

🔢 Parameters = the network's "memory"

When people say GPT-3 has "175 billion parameters," they mean 175 billion weights and biases that were learned from data. More parameters → more capacity to learn complex patterns.

A Simple Network in Code

import numpy as np

# Simple 2-layer neural network (numpy, no frameworks)
def sigmoid(z):
    return 1 / (1 + np.exp(-z))

# Initialize weights randomly
np.random.seed(42)
W1 = np.random.randn(3, 2)  # 3 hidden neurons, 2 inputs
b1 = np.zeros((3, 1))
W2 = np.random.randn(1, 3)  # 1 output neuron, 3 hidden
b2 = np.zeros((1, 1))

def forward(x):
    # Hidden layer
    z1 = W1 @ x + b1
    a1 = sigmoid(z1)
    # Output layer
    z2 = W2 @ a1 + b2
    a2 = sigmoid(z2)
    return a2, a1

# Run with input [1.0, 0.5]
x = np.array([[1.0], [0.5]])
output, hidden = forward(x)
print(f"Output: {output[0,0]:.4f}")

Universal Approximation

Here's a mind-blowing fact: a neural network with even one hidden layer (given enough neurons) can approximate any continuous function. This is the Universal Approximation Theorem.

In practice, deeper networks (more layers) are more parameter-efficient than wider networks (more neurons per layer). That's why deep learning took off — depth gives you expressive power without needing exponentially more neurons.

✅ Key Takeaways:
  • Neural networks = layers of neurons stacked together
  • Data flows forward: input → hidden layers → output
  • Each layer transforms the data using weights, biases, and activation
  • In matrix form: a = activation(Wx + b)
  • More layers = more abstract features learned
